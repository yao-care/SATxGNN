# SATxGNN - المملكة العربية السعودية: توقعات إعادة توظيف الأدوية

[![Website](https://img.shields.io/badge/Website-satxgnn.yao.care-blue)](https://satxgnn.yao.care)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

توقعات إعادة توظيف الأدوية (drug repurposing) للمملكة العربية السعودية باستخدام نموذج TxGNN.

## إخلاء المسؤولية

- نتائج هذا المشروع مخصصة لأغراض البحث فقط ولا تشكل نصيحة طبية.
- يتطلب مرشحو إعادة توظيف الأدوية التحقق السريري قبل التطبيق.

## نظرة عامة على المشروع

| العنصر | العدد |
|--------|-------|
| **تقارير الأدوية** | 808 |
| **إجمالي التوقعات** | 12,123,340 |

## طرق التوقع

### طريقة الرسم البياني المعرفي (Knowledge Graph)
استعلام مباشر عن العلاقات بين الأدوية والأمراض في الرسم البياني المعرفي لـ TxGNN، لتحديد المرشحين المحتملين لإعادة التوظيف بناءً على الروابط الموجودة في الشبكة الطبية الحيوية.

### طريقة التعلم العميق (Deep Learning)
يستخدم نموذج الشبكة العصبية المُدرَّب مسبقًا من TxGNN لحساب درجات التوقع، وتقييم احتمالية المؤشرات العلاجية الجديدة للأدوية المعتمدة.
